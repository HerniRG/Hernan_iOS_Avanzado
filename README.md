
# Hernán iOS Avanzado

## Descripción del Proyecto
Hernán iOS Avanzado es una aplicación que permite a los usuarios explorar héroes del universo Dragon Ball, mostrando sus detalles, transformaciones y localizaciones en un mapa. La app incluye funcionalidades como inicio de sesión, almacenamiento seguro de tokens y gestión de datos a través de Core Data.

## Funcionalidades
- **Splash Screen**: Pantalla de inicio atractiva.
- **Login**: Autenticación de usuarios utilizando JWT. Se guardan tokens de forma segura usando Keychain.
- **Pantalla Principal**: Muestra una lista de héroes con opciones de búsqueda y ordenación.
- **Detalles del Héroe**: Información sobre el héroe, incluyendo su imagen, descripción y transformaciones.
- **Mapa**: Visualiza las localizaciones de los héroes utilizando MapKit y MKARoundScene para vistas en 3D de las localizaciones.
